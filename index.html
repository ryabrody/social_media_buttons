Why using link href?

  • nice formatting
  • no need of additional javascript files
  • same button for each language (cause no text inside button)
  • no load-jumping

Why not using link href?

  • some troubles with empty spaces which has to be detected first (mail, google+,)
  • window which opens is not that nice
  • ( link solution is hacky (not supported by provider) )
